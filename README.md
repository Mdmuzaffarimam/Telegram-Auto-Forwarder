# Telegram Auto Forwarder (Bot Login System)

## Features
- Bot based interface
- User account login (OTP + 2FA)
- Incoming / Outgoing chats
- Media + Text forwarding
- Private & Restricted channels supported

## Setup
1. Fill API_ID, API_HASH, BOT_TOKEN in config.py
2. Install requirements
3. Run bot.py

## Commands
/start
/authorize
/incoming
/outgoing
/work
